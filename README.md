# NeuralChat 🤖

A clean, dark-themed AI chat interface powered by Claude — built with pure HTML, CSS & JavaScript. No frameworks, no dependencies.

## ✨ Features

- 🧠 **Real-time AI responses** powered by Claude (claude-sonnet-4)
- 💾 **Persistent chat memory** — conversations saved across sessions via localStorage
- 💻 **Code formatting** with syntax highlighting
- ⌨️ **Typing indicators** for a natural chat feel
- 💡 **Suggestion chips** to get started quickly
- 📱 **PWA ready** — installable on desktop & mobile like a native app
- 🌙 **Dark theme** with terminal-inspired design

## ⚙️ How to Run Locally

1. Clone the repo
```bash
git clone https://github.com/gorickroot/Neuralchat.git
cd Neuralchat
```

2. Open `index.html` in your browser — that's it. No build step, no npm install.

> ⚠️ To enable AI responses, you need an Anthropic API key. Due to browser CORS restrictions, direct API calls require either a backend proxy or a CORS-enabled service.

---

## 📦 Install as App (PWA)

1. Deploy to Netlify or GitHub Pages
2. Open the live URL in **Chrome or Edge**
3. Click the **install icon** in the address bar
4. NeuralChat installs like a native app on your device ✅
